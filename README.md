# Server Status

Скрипт для проверки состояния серверов:
- проверка свободного места на локальных дисках
- проверка необходимости перезагрузки после установки обновлений средствами SCCM
- проверка состояния сервисов VMware
- проверка состояния сервисов MS SQL
- проверка состояния сервисов MS TMG
- проверка состояния сервисов SurfCop
- проверка текущих алертов vCenter
- проверка размера локальной БД Proxy Inspector
 
Список серверов загружается из CSV-файла.

Формат файла: Location,Description,Name,Roles
- Location - местоположение сервера
- Description - понятное имя сервера
- Name - DNS-имя или IP-адрес сервера
- Roles - список ролей, разделенных пробелами (определяет что будет проверяться на этом сервере)

Возможные значения ролей:
- windows
- vmware_services
- mssql_services
- mstmg_services
- surfcop_services
- vcenter_alerts
- pi_local_db

Результаты сохраняются в HTML-файл. Используется Bootstrap v3.3.5.

# Требования

- Powershell 2.0
- Удаленный доступ к WMI на проверяемых серверах (проверка свободного места на дисках, необходимость перезагрузки, состояние сервисов)
- Доступ к системной шаре C$ (проверка размера файла)
- vSphere PowerCLI (проверка алертов vCenter)

# Использование

- Заполнить файл Servers.csv.
- Запустить файл Run_Script.cmd.
- Результат выполнения будет записан в файл HTML\server-status.html.
